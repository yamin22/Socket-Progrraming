Usage of the mchat.c

This program is for group chat on linux machine.

Compile the program using the gcc on linux machine and provide the complied program name as <program name>,
-mcip: give the ip of your machine to create a multicast group so others can join the group,
-port: port number on which you want to create the app, must be above the 1024

usange: ./<programe name> -mcip <IP address> -port <port number>
